Goal

The goal of this project is to access Icesat2's canopy data (ATL08) in a simple way.

User Interface

The UI is pretty much self-explanatory. To display ATL08 aggregation points, make sure the ATL08 button is switched on and zoom in.

There is no data in my area?

Only a small part of the planet data has been processed. If you would like to add a location, just drop me an email specifying the location e.g. city and country name , or coordinates at bruno.carle-icesat2 it should be ready in a couple of days.
